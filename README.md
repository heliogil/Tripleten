# Dashboard de Anúncios de Veículos

Este projeto é um aplicativo web desenvolvido com Streamlit, pandas e Plotly Express para explorar dados de anúncios de veículos usados.

## Funcionalidades

- Visualização de um histograma da quilometragem dos veículos.
- Visualização de um gráfico de dispersão comparando preço e quilometragem.
- Interface simples com caixas de seleção para exibir os gráficos.

## Como executar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
